BraillePDF API

Esta plataforma permite a maestros convertir archivos PDF a texto Braille, mediante procesamiento OCR y traducción automatizada. El resultado se puede enviar a un dispositivo externo como el ESP32. Administra usuarios, protege los accesos y traduce con facilidad.
